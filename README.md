# Tree Data Structure
Trees as Abstract Data Type in Java

## Usage

```sh
Usage: MainClass [options]
  Options:
    -h, --help

  * -i, --input
      The input
      Default: []
    -t, --type
      The type of the tree
      Default: BST
    -gui
      Show the visualization
      Default: false
```

## Example

```sh
java -jar -i 15,10,20,8,12,16,25 -gui
```

8, 3, 10, 1, 6, 4, 7, 14, 13
